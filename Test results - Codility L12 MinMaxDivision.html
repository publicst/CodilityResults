1. MinMaxDivision
Divide array A into K blocks and minimize the largest sum of any block.
Task description

You are given integers K, M and a non-empty zero-indexed array A consisting of N integers. Every element of the array is not greater than M.

You should divide this array into K blocks of consecutive elements. The size of the block is any integer between 0 and N. Every element of the array should belong to some block.

The sum of the block from X to Y equals A[X] + A[X + 1] + ... + A[Y]. The sum of empty block equals 0.

The large sum is the maximal sum of any block.

For example, you are given integers K = 3, M = 5 and array A such that:

  A[0] = 2
  A[1] = 1
  A[2] = 5
  A[3] = 1
  A[4] = 2
  A[5] = 2
  A[6] = 2

The array can be divided, for example, into the following blocks:

  • [2, 1, 5, 1, 2, 2, 2], [], [] with a large sum of 15;
  • [2], [1, 5, 1, 2], [2, 2] with a large sum of 9;
  • [2, 1, 5], [], [1, 2, 2, 2] with a large sum of 8;
  • [2, 1], [5, 1], [2, 2, 2] with a large sum of 6.

The goal is to minimize the large sum. In the above example, 6 is the minimal large sum.

that, given integers K, M and a non-empty zero-indexed array A consisting of N integers, returns the minimal large sum.

  A[0] = 2

  A[2] = 5
  A[3] = 1

  • N and K are integers within the range [1..100,000];
  • M is an integer within the range [0..10,000];
  • each element of array A is an integer within the range [0..M].

using System.Linq;
// you can also use other imports, for example:
// using System.Collections.Generic;

// you can use Console.WriteLine for debugging purposes, e.g.
// Console.WriteLine("this is a debug message");

class Solution {
    public int solution(int K, int M, int[] A) {
        // note to myself K is blocks we can fill
        int max = A.Sum();
        int min = A.Max(); // could use M but problem does not state its going to use M as element its just "possible" max I want to know real max
        int result = 0;    // if all elements are 0 in A then results are going to be 0
        
        if (K == 1) return max; // if theres only 1 block the result is going to be max
        if (K >= A.Length) return min; // if every element can be stored in separate block the result is going to be min
        
        while (min <= max)
        {
            int mid = (max + min) / 2;
            
            int blockNum = 1;
            int curBlockSum = A[0];
            for (int i = 1; i < A.Length; i++)
            {
                if (curBlockSum + A[i] > mid)
                {
                    blockNum++;
                    curBlockSum = A[i];
                }
                else
                {
                    curBlockSum += A[i];
                }
            }
            
            if (blockNum <= K) // we can lower result
            {
                max = mid - 1; // - 1 since we proved with above forloop and if statement that mid is going to fulfill the requirement
                result = mid;
            }
            else
            {
                min = mid + 1; // + 1 since we proved with above forloop and if statement that mid is NOT going to fulfill the requirement
                // hence no update on result either
            }
        }
        
        return result;
    }
}

using System.Linq;
// you can also use other imports, for example:
// using System.Collections.Generic;

// you can use Console.WriteLine for debugging purposes, e.g.
// Console.WriteLine("this is a debug message");

class Solution {
    public int solution(int K, int M, int[] A) {
        // note to myself K is blocks we can fill
        int max = A.Sum();
        int min = A.Max(); // could use M but problem does not state its going to use M as element its just "possible" max I want to know real max
        int result = 0;    // if all elements are 0 in A then results are going to be 0
        
        if (K == 1) return max; // if theres only 1 block the result is going to be max
        if (K >= A.Length) return min; // if every element can be stored in separate block the result is going to be min
        
        while (min <= max)
        {
            int mid = (max + min) / 2;
            
            int blockNum = 1;
            int curBlockSum = A[0];
            for (int i = 1; i < A.Length; i++)
            {
                if (curBlockSum + A[i] > mid)
                {
                    blockNum++;
                    curBlockSum = A[i];
                }
                else
                {
                    curBlockSum += A[i];
                }
            }
            
            if (blockNum <= K) // we can lower result
            {
                max = mid - 1; // - 1 since we proved with above forloop and if statement that mid is going to fulfill the requirement
                result = mid;
            }
            else
            {
                min = mid + 1; // + 1 since we proved with above forloop and if statement that mid is NOT going to fulfill the requirement
                // hence no update on result either
            }
        }
        
        return result;
    }
}
